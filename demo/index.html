🔒 Bell-LaPadula Rules

↑
No Read Up
Can't read above your clearance
↓
No Write Down
Can't send info to lower clearance

3 Lines. That's It.

Add BLP security to your AI agents in seconds.

example.py 
from clearance.checker import create_checker
from clearance import SecurityLevel, User

# Setup with security keywords
checker = create_checker({
    "revenue": SecurityLevel.EXECUTIVE,
    "salary": SecurityLevel.MANAGER,
})

# Check before sending
staff = User("u1", "Alice", SecurityLevel.STAFF)
result = checker.check_write("Q3 revenue is $10M", staff)

if not result.allowed:
    print(f"Blocked: {result.violation}")  # NO_WRITE_DOWN
