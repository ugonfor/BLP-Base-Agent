## 설계 원칙

1. **단순성**: BLP 모델의 두 규칙만 적용 (No Read Up, No Write Down)
2. **확장성**: 프로토콜 기반 설계로 커스텀 구현 가능
3. **테스트 용이성**: 모든 컴포넌트가 의존성 주입 지원
4. **무상태**: 핵심 체커는 상태를 가지지 않음
